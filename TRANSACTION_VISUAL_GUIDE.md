## 🎨 Цветовая схема

### Кнопка транзакций:
- Фон: Оранжевый градиент `#ff9800 → #f57c00`
- Текст: Белый
- Иконка: 💰 (fa-wallet)

### Пополнение (deposit):
- Цвет: Зеленый `#4caf50`
- Иконка: ⬇️ (fa-arrow-down)
- Знак: `+`

### Вывод (withdrawal):
- Цвет: Красный `#ea4335`
- Иконка: ⬆️ (fa-arrow-up)
- Знак: `-`

## 📱 Пример использования

### Для пользователя:
1. Открыть приложение
2. Нажать на оранжевую кнопку "💰 Транзакции"
3. Увидеть историю всех пополнений и выводов
4. Закрыть окно кнопкой ×

### Для администратора:
1. Открыть админ панель (кнопка в профиле)
2. Прокрутить до секции "💰 Управление транзакциями"
3. Ввести имя пользователя (например: "Fronz")
4. Выбрать тип: "Пополнение" или "Вывод"
5. Ввести сумму (например: 10000)
6. Нажать "➕ Добавить транзакцию"
7. Увидеть уведомление об успехе
8. Пользователь увидит транзакцию в своей истории
